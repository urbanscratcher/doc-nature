# 2. 뉴턴의 운동 2법칙

## 뉴턴의 운동 2법칙

- 힘 = 질량 X 가속도
- 가속도 = 힘 / 질량
- 물체에 가하는 힘이 커질 수록 가속도도 커지고, 빠른 속도로 움직일 것임
- 그런데 물체가 무겁다면 가속도가 작아지기 때문에 느리게 움직일 것임

:::note[무게와 질량]

- 질량(mass): 어떤 물체가 가지고 있는 물질의 양 ($kg$)
- 무게(weight): 중력이 해당 물체에 얼마나 큰 힘을 작용하는지 의미. ($N$)
- 밀도(density): 특정한 부피에서의 질량 ($kg/m^3$)
  :::
